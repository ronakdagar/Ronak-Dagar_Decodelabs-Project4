📊 Project Title: Data Visualization

In this project, I created powerful visual representations of a real-world Vehicle Dataset to communicate data insights clearly and effectively by:

✔ Created a Bar Chart to compare the count of vehicles by brand/make

✔ Plotted a Histogram to visualize the distribution of vehicle prices and mileage

✔ Built a Box Plot to detect outliers in price across different fuel types

✔ Designed a Scatter Plot to explore the relationship between mileage and price

✔ Used a Heatmap to visualize correlations between numerical features like price, mileage, and engine size

✔ Created a Pie Chart to show the proportion of vehicles by fuel type (Petrol, Diesel, Electric, etc.)

✔ Plotted a Line Graph to analyze price trends across different vehicle years

✔ Chose the most appropriate chart type for each insight to maximize clarity and impact

✔ Highlighted key findings and patterns through well-labeled and styled visualizations

🔍 Key Outcome:
The visualizations clearly revealed:
✅ Strong negative correlation between mileage and vehicle price

✅ Petrol and Diesel dominate the vehicle market by fuel type

✅ Newer vehicles tend to have significantly higher price values

✅ Several high-mileage vehicles were identified as major outliers in pricing
